# College Mapping Website

A full-stack web application for campus navigation, featuring secure authentication,
an interactive map, and an AI-powered chatbot assistant.

## Features
- 🔐 Google OAuth authentication
- 🗺️ Interactive campus map built with Leaflet.js
- 🤖 AI chatbot powered by Ollama for campus queries
- 🗄️ PostgreSQL-backed data storage

## Tech Stack
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Frontend:** HTML, CSS, JS, Bootstrap5
- **Auth:** Google OAuth 2.0
- **AI:** Ollama (local LLM)

## Getting Started

### Prerequisites
- Node.js (v20.11.0 or higher)
- PostgreSQL
- Ollama installed locally

### Installation
\`\`\`bash
git clone https://github.com/yourusername/college-mapping-website.git
cd college-mapping-website
npm install
\`\`\`

### Environment Variables
## Environment Variables
Copy `.env.example` to `.env` and fill in your own credentials.

### Run
\`\`\`bash
node "index.js"
\`\`\`
